# 表情包生成以及微信平台上传流程
## 第一章 表情包生成流程
1. 登录图生图模型，比如即梦AI（https://jimeng.jianying.com）
![alt text]([.\wechat-sticker-publish-workflow\images\4.png](https://github.com/wsk12363/wechat-sticker-publish-workflow/blob/main/images/4.png?raw=true))
2. 上传一张小狗的图片，并使用提示词生成表情包
> 第一段提示词：请分析我上传的这只小狗的照片，提炼出它的关键外貌特征（品种、毛色、体型、耳朵、眼睛、嘴巴、尾巴等）。然后基于这些特征，用 简洁可爱的微信表情包手绘风格 重新绘制这只小狗，背景为纯白色，确保形象适合制作系列表情。以后所有表情都将基于这个标准形象生成。
![alt text](.\wechat-sticker-publish-workflow\images\3.png)

> 第二段提示词
请基于之前确定的“标准小狗形象”，为我创作一套共16个微信聊天表情包。 画风保持统一：简洁可爱手绘风，纯白背景，高清，适合微信表情平台（240x240px）。 每个表情图片需对应一个中文表情标签，请将标签文字以浅色小字置于图片角落，方便我后期发布。 以下是16个表情的列表，请按顺序逐个生成，每个表情需严格按照描述绘制： 1. 标签：开心 描述：小狗眯着眼睛，嘴角开心地向上咧开，舌头微微伸出，显得非常快乐。 2. 标签：委屈 描述：小狗耷拉着眉毛和耳朵，眼角有泪光，嘴巴向下撇成“ω”形，像被批评了一样。 3. 标签：疑问 描述：小狗歪着头，睁大眼睛，眉头微皱，头顶画一个问号，满是不解。 4. 标签：生气 描述：小狗眉头紧锁，嘴角向下，鼻子上方挤出皱纹，头上画冒火的符号，气得鼓鼓的。 5. 标签：害羞 描述：小狗脸红红，眼神羞涩地瞟向一边，用一只爪子挠挠头，另一只爪子垂着。 6. 标签：嘚瑟 描述：小狗抬起一只爪子挡在嘴前，眼神轻佻，面带得意的坏笑，像在嘲讽。 7. 标签：大哭 描述：小狗眼睛变成两个“X”形，张着嘴嗷嗷大哭，眼泪喷成两道小喷泉。 8. 标签：睡 描述：小狗侧趴在地上，闭眼，嘴角流着口水泡泡，表情安宁。 9. 标签：无语 描述：小狗半眯着眼睛，面无表情，额头画三条黑线或一滴汗，一副嫌弃的样子。 10. 标签：爱你 描述：小狗眼睛变成两个大爱心，微微笑着，爪子在胸前比一个爱心手势。 11. 标签：吃瓜 描述：小狗蹲坐着，用爪子捧着半个西瓜，嘴边有瓜子，眼神专注地看着前方。 12. 标签：收到 描述：小狗立正站好，敬个军礼，表情认真严肃，像接受任务。 13. 标签：溜了 描述：小狗只露出侧身和翘起的小尾巴，身后画几道烟尘线，表示快速逃跑。 14. 标签：好 描述：小狗竖起一只爪子的大拇指，闭眼微笑，非常肯定和认同。 15. 标签：摸头 描述：小狗伸出爪子，轻轻摸着自己的后脑勺，不好意思地傻笑。 16. 标签：再见 描述：小狗面对正前方，微笑，举起一只爪子挥动告别，表情友善。
